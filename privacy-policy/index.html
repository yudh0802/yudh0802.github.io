Privacy Policy

Last updated: April 15, 2026

Overview

This application ("Calendar MCP") is a personal utility that integrates with Google Calendar for personal schedule management. It is not intended for public use.

Data Collection

This application accesses Google Calendar data solely for the purpose of creating, reading, updating, and deleting calendar events on behalf of the authenticated user.

Data Usage

  • Calendar data is accessed only during active use of the application.
  • No data is stored on external servers.
  • No data is shared with third parties.
  • All data processing occurs locally on the user's device.

Data Retention

This application does not retain any Google Calendar data beyond the active session. Authentication tokens are stored locally on the user's device.

Third-Party Services

This application uses Google OAuth 2.0 for authentication. Please refer to Google's Privacy Policy for information on how Google handles your data.

Contact

For questions about this privacy policy, please contact the developer via GitHub.
